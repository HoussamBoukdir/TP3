Explication des éléments HTML

  • !DOCTYPE html: Définit la version du HTML (HTML5 dans ce cas).
  • html: Conteneur principal qui englobe tout le contenu de la page.
  • head: Contient les méta-informations sur le document (titre, encodage, liens vers les CSS, scripts, etc.).
  • title: Définit le titre de la page qui apparaît dans l'onglet du navigateur.
  • meta: Fournit des métadonnées comme la description de la page, le jeu de caractères, etc.
  • link: Lien vers des ressources externes comme des fichiers CSS.
  • script: Intègre ou lie des scripts JavaScript.
  • section: Section de la page.
  • div: division de bloc html.

    • Balises de Format

    • <h1> ; >>> <h6>: Définit les titres, <h1> étant le plus important, <h6> le moins.
    • <p>: Définit un paragraphe.
    • <br>: Retour à la ligne.
    • <hr>: Ligne horizontale.
    • <strong>: Texte en gras.
    • <em>:emphasized text
    • <b>: Met le texte en gras.
    • <i>: Met le texte en italique.
    • <u>: ?

    TABLEAUX

    • <table>: Crée un tableau.
    • <tr>: Définit une ligne dans le tableau.
    • <th>: Définit une cellule d'en-tête.
    • <td>: Définit une cellule dans une ligne.
    • <thead>: Groupe les lignes d'en-tête d'un tableau.
    • <tbody>: Groupe les lignes de contenu d'un tableau.
    • <tfoot>: Groupe les lignes de pied de tableau.

    Listes

    • <ul>: Liste à puces (non ordonnée).
    • <ol>: Liste ordonnée (numérotée).
    • <li>: Élément d'une liste.

    Formulaires

    • <form>: Formulaire.
    • <input>: Champ d'entrée de formulaire (texte, mot de passe, email, etc.).
    • <textarea>: Zone de saisie multiligne.
    • <button>: Bouton d'envoi ou autre action.
    • <select>: Liste déroulante.
    • <option>: Élément d'une liste déroulante.
    • <label>: Libellé d'un champ.

    Balise de multimedia

    • <audio> : Permet d’intégrer un contenu audio (musique, effets sonores, etc.).
    • <video> : Permet d’intégrer un contenu vidéo (films, tutoriels, etc.).
    • <iframe> : Permet d’intégrer un contenu externe (page web, carte, vidéo, etc.).
    • <source> : Spécifie la source d’un fichier multimédia pour les balises <audio> ou <video>.
